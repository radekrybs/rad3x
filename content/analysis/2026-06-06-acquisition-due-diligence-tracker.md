This is a working **due-diligence register** for a medical-building acquisition:
50+ items spanning transaction & legal, zoning & planning, building code & fire,
structural, electrical, mechanical, environmental & hazmat, healthcare-regulatory
(CPSO OHP, pharmacy, lab), and financial workstreams.

It's interactive: set each item's status (Not started / In progress / Complete /
Red flag), add field notes, filter by status or category, search, and watch the
live progress stamp and per-category counters update. Each item carries the
"why it matters" rationale and a flagged owner and phase.

**Your status and note edits are saved automatically** and persist between
visits on this device.
